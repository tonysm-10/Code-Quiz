Coding Challenge

Some Rules of this Challenge

1. You have 75 seconds to answer all the questions
2. If you answer incorrectly you will lose 10 seconds off your time
3. When all the questions are answered or the timer reaches 0 the game is over
4. When the game is over the score you get is the time you had left
5. Answer the questions fast and get as few wrong as possible to get a highest score possible, have fun!
